Project  :  Node.js web application

Frameworks : Nodejs, express.js

Database : MongoDB

(Deployed) Online Link : https://nodejs-web-application.herokuapp.com/

Documentation : Swagger-Ui, link : "http://localhost:5000/api-docs"

    

    Instructions to run the code :

1. Download and extract the code
2. Go the source directory through the console
3. Install mongoDB compass to run the code locally
4. Then run 'npm install'  and run 'npm start'
